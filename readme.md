## 功能概覽

### 💨 左側快捷
- 一鍵狀態恢復 + PC 立刻高潮  

### 🏠 空間節點
- 三個常駐節點：
  - 孤兒院臥室  
  - 伊甸家（未觸發劇情建議別直接傳送）  
  - 隨身衣櫃  
- 原作者: [用户_aeVUUeW@貼吧-DOL吧][YL+p_wardrobe]  

### ✨ 言靈系統
- 支援二 / 三代言靈，左側即時顯示  
- 可同時新增多個言靈  
- 調用遊戲內 `widget` / `JS`  
- 優化 By 北极星勾陈一  

### ~~🏗 場景創建~~
- ~~遊戲內自定義場景，言靈作入口~~  
- ~~核心代碼 By Tony70124[bccme]~~  
- 已移除

### ⚔ 戰鬥 / 屬性控制
- 戰鬥傷害加倍 / 戰鬥疼痛衰減  
- 戰鬥中敵人狀態顯示
    - 簡易的圖像化血條顯示
    - 動態插入UI，兼容各類敵人狀態顯示模組或整合包  
     
- **角色屬性控制面板**：
  - 可調整：疼痛 / 創傷 / 控制值 / 壓力 / 敏感度 / 性興奮 / 疲勞 / 技能  
  - 每個屬性支援正負倍率與開關  
- **純潔永駐**：
    - 可設定純潔最大值，即使不再是處也能滿純潔  
    - 調整純潔增加及減少速度    

### 🍼 擠奶 / 大爆🐍
- **大量擠🥛模式**：
  - 一次榨乾容量 6000 🥛
  - 可修改當前可儲存🥛量
  - 可修改當前🥛量最大值
  - 可修改當前有多少🥛
- **大爆🐍模式**：
  - 與大量擠🥛功能雷同，但數值更大
  - 可修改當前可儲存🍼量、最大值與總量
  - 僅男Pc / Futa Pc 或啟用 DEBUG 模式才會顯示此功能按鈕

### 💰 商業 / 收支功能
- 黑心商店模式：
  - 一鍵清空庫存  
  - 銷售金額加倍  
- 跳舞報酬加倍  
- 尋歡洞報酬加倍  
- 收支倍率調整：
  - 收入倍率 1~100  
  - 支出倍率 0.1~1  

###⌛ 時間流速控制
- 控制時間流逝的速度
    - 時間停止  
    - 時間加倍流逝    

### 🎓 學習模式
- 不去學校不計旷課  
- 當週考試成功率固定 120%  
- 可手動拉滿當週考試成功率  

### 🛠 DEBUG 模式
- 慎用 開啟原版的除錯模式 

### 👗 服裝功能
- 一鍵全添加 / 指定添加 / 關鍵字添加
- 特殊服裝（項圈、口球等）未排除  

### 🪄 魔術迴路
- 自訂義紋身文字 / 屬性  
- 可輸入言靈  

### 🌡 自動恆溫（測試中）
- 服裝保暖視氣溫自動調節  

### 🧬 Pc 懷孕功能
- 自訂非生物懷孕（觸手等）  
- 自訂人類懷孕（可強制懷上 NPC 孩子）  
- 修改寄生物懷孕插槽數量（例如前後各懷10個蒼白觸手）
    * 可修改懷孕機率倍數，適應更多插槽

### ⚙️ 其他
- 強制顯示作弊按鈕  
- 強制啟用成就系統
