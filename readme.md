### ⚔ 戰鬥 / 屬性控制
- **戰鬥傷害加倍 / 戰鬥疼痛衰減**  
- **戰鬥中敵人狀態顯示**
    - 簡易的圖像化血條顯示
    - 動態插入UI，兼容各類敵人狀態顯示模組或整合包  
     
- **角色屬性控制面板**：
  - 可調整：疼痛 / 創傷 / 控制值 / 壓力 / 敏感度 / 性興奮 / 疲勞 / 技能  
  - 每個屬性支援正負倍率與開關  
- **純潔永駐**：
    - 可設定純潔最大值，即使不再是處也能滿純潔  
    - 調整純潔增加及減少速度    

### 🍼 擠奶 / 大爆🐍
- **大量擠🥛模式**：
  - 一次榨乾容量 6000 🥛
  - 可修改當前可儲存🥛量
  - 可修改當前🥛量最大值
  - 可修改當前有多少🥛
- **大爆🐍模式**：
  - 與大量擠🥛功能雷同，但數值更大
  - 可修改當前可儲存🍼量、最大值與總量
  - 僅男Pc / Futa Pc 或啟用 DEBUG 模式才會顯示此功能按鈕

### 👊 新增戰鬥動作
  * 天使化
    
    - 聖光：消耗一次聖光璀璨，對所有敵人造成其當前生命值三分之一的傷害，並提升敵方憤怒、降低敵方信任。
    - 聖療：恢復自身狀態，降低疼痛、壓力與創傷，並恢復少量自控力。
    - 神聖治癒：消耗一次聖光璀璨，大幅恢復身心狀態，淨化部分負面效果並恢復自控力。
    - 祈禱：每場戰鬥限用一次，恢復一次聖光璀璨，清空雙方興奮度，恢復所有敵人少量生命值並舒緩自身狀態。
    - 神聖屏障：每場戰鬥限用一次，消耗一次聖光璀璨，擊退接近插入的目標，降低自身興奮度並恢復自控力，同時提升敵方憤怒並降低敵方信任。

    - 聖潔庇護（觸手戰）：消耗天使轉化，擊退接近入口的觸手，不受高興奮度施法失敗影響。
    - 光耀驅邪（觸手戰）：消耗天使轉化，驅離指定觸手。
    - 聖痕（觸手戰）：消耗天使轉化，對指定觸手造成傷害並施加聖痕，強化後續神聖技能。
    - 聖裁（觸手戰）：消耗天使轉化，重創指定觸手並打斷其行動。
    - 聖光爆發（觸手戰）：消耗大量天使轉化，對所有觸手造成傷害並打斷其行動。
    - 祈禱（觸手戰）：恢復天使轉化與聖光璀璨，但會增加疲勞與壓力。

  * 惡魔化
    
    - 淫蝕：侵蝕敵人的意志，造成少量傷害並提升敵方興奮度，自身也會逐漸興奮。
    - 惡魔之觸：將敵人的慾望化為傷害，造成傷害並提升敵方興奮度，同時增加自身興奮、降低自控力，並提升敵方憤怒、降低敵方信任。
    - 痛苦轉嫁：每場戰鬥限用一次，將自身累積的痛苦、壓力、創傷與疲勞轉化為傷害反噬插入者，恢復自身狀態，並大幅提升自身興奮度。
    - 沉淪：主動擁抱魅魔本能，降低純潔以換取力量，提升敵方興奮度並造成少量傷害。
    - 墮慾衝擊：將失去的純潔轉化為力量攻擊所有敵人，純潔越低傷害越高，但會增加自身興奮、降低自控力，並提升敵方憤怒、降低敵方信任。

    - 深淵汲取（觸手戰）：吸收插入觸手的生命恢復自身狀態與自控力，但會大幅提升自身興奮度。
    - 淫慾供奉（觸手戰）：將自身興奮度化為養分供奉觸手，大幅恢復其生命值並提升生命上限，但會降低自身自控力。
    - 肉慾轉嫁（觸手戰）：犧牲插入中的觸手，將其生命力轉移至其他觸手並施加惡魔標記。
    - 惡魔回收（觸手戰）：回收惡魔標記中的力量，恢復惡魔轉化並重創被標記的觸手。

  * 貓化
    
    - 貓爪攻擊：對指定目標造成其最大生命值 10% 的傷害，並略微提升敵方憤怒、降低敵方信任。
    - 呼嚕：安撫敵人，降低憤怒、提升信任，同時舒緩自身狀態。
    - 理毛：整理儀容，降低壓力、疲勞與創傷，恢復少量自控力。
    - 舔舐：大幅提升敵方興奮度，並略微提升信任。
    - 發情：每場戰鬥限用一次，大幅提升敵方興奮度，同時增加自身興奮、降低自控力，並略微提升敵方信任。

    - 貓爪（觸手戰）：對指定觸手造成其當前生命值一定比例的傷害。
    - 呼嚕（觸手戰）：舒緩自身狀態，同時干擾所有觸手的行動。
    - 貓之柔韌（觸手戰）：以靈活身體閃避觸手，打斷即將插入的觸手並恢復自控力。

  * 通用
    
    - 夾緊：提升雙方興奮度，非自願狀態下會對目標造成少量傷害。
    - 絞緊：每場戰鬥限用一次，重創目標並強制解除插入狀態，同時大幅增加敵方憤怒、降低敵方信任與興奮度，並增加自身疼痛與興奮度。
